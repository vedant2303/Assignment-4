# Assignment-4
This is a C++ programme has 1 cpp file and 1 header file. 
This program basically this function performs shakersort algorithm and reports the time in both 
ascending and descending order.

The following is what the program will do
Util.h includes three functions:
  1) First function takes an integer and returns an array of size n filled with random numbers
  2) Second function takes array, it's size and sorttype. If sorttype is 0, it sorts in 
     ascending order, if sorttype is 1, then it sorts in descending order.
  3) Third function prints the array to console.

ShakerSort.cpp includes:
  1) This file contains the main() function whose function is to perform sort and returns the 
     time of ascending and descending order.
